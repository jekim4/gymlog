## GymLog (가칭) — MVP 기능 정의 표

| 기능 ID | 대분류 | 중분류 | 기능명 | 기능 상세 정의 | 화면명 | 중요도 |
|---|---|---|---|---|---|---|
| F-001 | 세션 | 조회 | 운동 세션 목록 조회 | 사용자는 운동 세션 목록을 조회할 수 있으며, 목록은 운동일 기준 최신순으로 정렬되고 각 항목에 운동일·종목 수·세트 수가 표시된다. | 세션 목록 화면 | High |
| F-002 | 세션 | 조회 | 운동 세션 상세 조회 | 사용자는 세션 목록에서 특정 세션을 선택해, 해당 세션의 운동일·제목·메모와 함께 종목별 세트 정보(무게·횟수·세트 순번)를 조회할 수 있다. | 세션 상세 화면 | High |
| F-003 | 세션 | 등록 | 운동 세션 등록 | 사용자는 새로운 운동 세션을 생성할 수 있으며, 최소 운동일을 입력하고(기본값: 오늘) 선택적으로 제목과 메모를 입력한 뒤 빈 세션을 저장한다. | 세션 생성 화면 | High |
| F-004 | 세션 | 등록 | 세션 내 종목/세트 등록 | 사용자는 세션 상세에서 부위별로 그룹화된 운동 종목을 선택하거나 추가하고, 선택한 종목에 대해 무게와 반복 횟수를 입력하여 세트를 한 줄씩 등록할 수 있다. | 세션 상세 화면 | High |
| F-005 | 세션 | 수정 | 세션 정보 수정 | 사용자는 기존 세션의 운동일·제목·메모를 수정할 수 있으며, 수정 내용은 저장 후 세션 상세와 목록에 반영된다. | 세션 상세 화면 | Medium |
| F-006 | 세션 | 수정 | 세트 정보 수정/삭제 | 사용자는 세션 상세에서 특정 종목의 세트 정보를 선택해 무게·반복 횟수를 수정하거나 세트를 삭제할 수 있으며, 변경 결과가 즉시 저장된다. | 세션 상세 화면 | High |
| F-007 | 세션 | 삭제 | 운동 세션 삭제 | 사용자는 더 이상 필요 없는 세션을 삭제할 수 있고, 삭제 시 확인 메시지를 거치며 해당 세션과 그 안의 모든 종목·세트 정보가 DB에서 물리적으로 제거된다. | 세션 목록 화면 | Medium |
| F-008 | 종목 | 조회 | 운동 종목 목록 조회 | 사용자는 등록된 운동 종목 목록을 부위별 그룹 형태로 조회할 수 있으며, 각 종목의 이름·부위·브랜드/기구 구분 정보를 확인할 수 있다. | 운동 종목 목록 화면 | High |
| F-009 | 종목 | 등록 | 운동 종목 등록 | 사용자는 새로운 운동 종목을 등록할 수 있으며, 최소 종목 이름과 부위를 입력하고 선택적으로 브랜드/기구 구분을 입력하여 저장할 수 있다. | 운동 종목 등록 화면 | High |
| F-010 | 종목 | 초기데이터 | 기본 종목 데이터 제공 | 시스템은 초기 실행 시 스쿼트·데드리프트·벤치프레스 등 일부 기본 프리웨이트 종목을 내장 데이터로 제공하며, 사용자는 이를 수정 없이 바로 사용할 수 있다. | 기본 종목 데이터 제공 | Medium |
