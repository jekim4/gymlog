## GymLog (가칭) — 와이어프레임 템플릿

### 사용 방법
- 각 화면 섹션을 복사해 화면별로 작성한다.
- MVP에서는 저충실도(텍스트/박스) 수준으로 먼저 완성한다.
- 각 화면에 대응되는 기능 ID(`F-001` 등)를 반드시 연결한다.

---

## 화면 1. 세션 목록 화면

### 1) 화면 목적
- 사용자가 최근 운동 기록을 빠르게 확인하고, 상세로 진입할 수 있게 한다.

### 2) 연결 기능 ID
- `F-001`, `F-007`

### 3) 핵심 UI 요소
- 상단 제목: `운동 세션 목록`
- 세션 카드/리스트
  - 운동일
  - 세션 제목(기본값: `YYYY-MM-DD 운동`, 수정 가능)
  - 종목 수
  - 세트 수
- 정렬 기준 표시: 최신순
- `새 세션 만들기` 버튼

### 4) 사용자 액션
- 세션 항목 클릭 → 세션 상세 이동
- 새 세션 만들기 클릭 → 세션 생성 화면 이동
- 세션 삭제(옵션 메뉴) 실행

### 5) 예외/정책
- 세션 전체 삭제는 확인 메시지 1회 권장
- 데이터가 없으면 빈 상태 메시지 표시

### 6) 메모(레이아웃 스케치)
- [상단 헤더]
  - 제목: 운동 세션 목록
  - 우측 아이콘: 종목 관리 이동
- [정렬 바]
  - 최신순 고정 문구(초기 MVP)
- [세션 리스트]
  - 카드 1: 2026-04-08 / 2026-04-08 운동 / 종목 7개 / 세트 24개
  - 카드 2: 2026-04-06 / 푸시 데이 / 종목 6개 / 세트 20개
  - 카드 우측 더보기: 삭제
- [하단 고정 CTA]
  - + 새 세션 만들기
- [빈 상태]
  - 아직 기록된 세션이 없습니다.
  - 첫 세션 만들기 버튼

## 화면 2. 세션 상세 화면

### 1) 화면 목적
- 선택한 세션의 종목/세트를 조회하고, 수정/삭제를 수행한다.

### 2) 연결 기능 ID
- `F-002`, `F-004`, `F-005`, `F-006`

### 3) 핵심 UI 요소
- 세션 헤더
  - 운동일
  - 세션 제목(수정 가능)
  - 메모(선택)
- 종목 블록 리스트(예: 해머 인클라인 프레스)
- 세트 리스트
  - 기본 표기: 세트 1행 저장
  - 화면 표기: 같은 무게/같은 횟수 연속 시 `n세트 묶음` 표기 가능
- `종목 추가` 버튼
- `세트 추가` 버튼

### 4) 사용자 액션
- 제목/메모 수정
- 종목 블록 추가/삭제
- 세트 추가/수정/삭제

### 5) 예외/정책
- 세션 내 종목/세트 삭제 시 확인 대화상자 생략(MVP)
- 세트 라벨(워밍업/피더/탑/백오프)은 MVP 필수 항목 아님

### 6) 메모(레이아웃 스케치)
- [상단 헤더]
  - 뒤로가기 / 날짜 / 더보기
- [세션 정보 카드]
  - 운동일: 2026-04-08
  - 제목: 푸시 데이 (수정 아이콘)
  - 메모: 가슴 + 어깨 중심, 탑세트 후 백오프
- [종목 블록 리스트]
  - 블록 A: 해머 아이소 레터럴 인클라인 프레스
    - Start: 3.6kg (표시용)
    - 세트 표시: 원판 0kg x 15회 x 1세트
    - 세트 표시: 원판 2.5kg(양쪽) x 15회 x 2세트
    - 세트 표시: 원판 0kg x 20회 x 1세트
    - 액션: +세트추가 / 종목삭제
  - 블록 B: 해머 아이소 레터럴 숄더프레스
    - Start: 4.5kg (표시용)
    - 세트 표시: 원판 0kg x 15회 x 3세트
    - 액션: +세트추가 / 종목삭제
- [하단 액션]
  - + 종목 추가
  - 세션 전체 삭제(더보기 메뉴)

## 화면 3. 세션 생성 화면

### 1) 화면 목적
- 새 운동 세션의 기본정보를 빠르게 생성한다.

### 2) 연결 기능 ID
- `F-003`

### 3) 핵심 UI 요소
- 운동일 입력(기본값: 오늘)
- 세션 제목 입력
  - 자동값: `YYYY-MM-DD 운동`
  - 사용자 수정 가능
- 메모 입력(선택)
- 저장 버튼

### 4) 사용자 액션
- 저장 후 세션 상세로 이동
- 취소 후 목록으로 복귀

### 5) 예외/정책
- 운동일은 필수
- 제목은 자동 채움(사용자 수정 가능)

### 6) 메모(레이아웃 스케치)
- [상단 헤더]
  - 제목: 새 세션 만들기
  - 취소 버튼
- [입력 폼]
  - 운동일: date picker (기본 오늘)
  - 제목: 기본값 자동 입력 `2026-04-08 운동`
  - 안내 문구: 제목은 자유 수정 가능 (예: 푸시 데이, 풀 데이)
  - 메모: multiline (선택)
- [하단 CTA]
  - 저장 후 세션 상세로 이동

---

## 화면 4. 운동 종목 목록/등록 화면

### 1) 화면 목적
- 운동 종목 라이브러리를 조회하고, 새 종목을 등록/삭제한다.

### 2) 연결 기능 ID
- `F-008`, `F-009`, `F-010`

### 3) 핵심 UI 요소
- 부위 탭/그룹
  - 가슴 / 등 / 어깨 / 팔 / 하체 / 코어
- 종목 리스트 항목
  - 종목명
  - 브랜드(선택)
  - Start(기본 저항) 표시(선택)
- 종목 등록 폼
  - 종목 이름(필수)
  - 부위(필수)
  - 브랜드/기구 구분(선택)
  - Start 정보(선택: 괄호/메모/필드)

### 4) 사용자 액션
- 종목 선택(세션 상세에서 호출 시)
- 종목 등록
- 종목 삭제

### 5) 예외/정책
- 종목 삭제 후에도 과거 세션의 종목명은 기록 당시 이름으로 유지
- 내장 기본 종목 삭제 허용 여부는 추후 정책으로 결정

### 6) 메모(레이아웃 스케치)
- [상단 헤더]
  - 제목: 운동 종목 라이브러리
  - + 종목 등록 버튼
- [부위 탭]
  - 가슴 | 등 | 어깨 | 팔 | 하체 | 코어
- [종목 리스트(가슴 탭 예시)]
  - 해머 인클라인 프레스 (Hammer Strength) / Start 3.6kg
  - 해머 와이드 체스트 (Hammer Strength) / Start 0.9kg
  - 라이프피트니스 체스트프레스 (Life Fitness)
  - 각 행 액션: 선택 / 삭제
- [종목 등록 바텀시트 or 별도 섹션]
  - 이름(필수), 부위(필수), 브랜드(선택)
  - Start 메모: 예) Start 8lbs/3.6kg
  - 저장 버튼
- [정책 안내]
  - 종목 삭제 시 과거 세션 기록의 이름은 유지됨
